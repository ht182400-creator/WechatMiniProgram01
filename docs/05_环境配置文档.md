## 2. Python 环境配置

### 2.1 安装 Python

**Windows:**
```bash
# 下载并安装 Python 3.11
# https://www.python.org/downloads/

# 验证安装
python --version
```

**Linux (Ubuntu/Debian):**
```bash
sudo apt update
sudo apt install python3.11 python3.11-venv python3.11-dev
python3.11 --version
```

**macOS:**
```bash
# 使用 Homebrew
brew install python@3.11

# 或使用 pyenv
brew install pyenv
pyenv install 3.11.0
pyenv global 3.11.0
```

### 2.2 创建虚拟环境

```bash
# 进入项目目录
cd WechatMiniProgram01

# 创建虚拟环境
python -m venv venv

# 激活虚拟环境
# Windows:
venv\Scripts\activate

# Linux/macOS:
source venv/bin/activate

# 验证激活
which python
```

### 2.3 安装依赖

```bash
# 激活虚拟环境后
cd backend

# 安装核心依赖
pip install --upgrade pip
pip install -r requirements.txt

# 验证安装
python -c "import fastapi; import pandas; import sklearn"
```

### 2.4 常用 pip 命令

```bash
# 查看已安装的包
pip list

# 导出依赖列表
pip freeze > requirements_lock.txt

# 从文件安装
pip install -r requirements.txt

# 安装特定版本
pip install fastapi==0.109.0
```

## 5. 数据库配置

```python

# 数据文件位于 backend/data/stock_quant.db
```

### 5.2 PostgreSQL (生产环境)

```bash
# 安装 PostgreSQL
# Ubuntu
sudo apt install postgresql postgresql-contrib

# macOS
brew install postgresql@15
brew services start postgresql@15

# 创建数据库和用户
sudo -u postgres psql
CREATE DATABASE stockquant;
CREATE USER squser WITH PASSWORD 'your_password';
GRANT ALL PRIVILEGES ON DATABASE stockquant TO squser;
```


## 7. Docker 配置 (可选)

### 7.1 后端 Dockerfile

```dockerfile
# backend/Dockerfile
FROM python:3.11-slim

WORKDIR /app

# 安装系统依赖
RUN apt-get update && apt-get install -y \
    build-essential \
    && rm -rf /var/lib/apt/lists/*

# 复制依赖文件
COPY requirements.txt .

# 安装 Python 依赖
RUN pip install --no-cache-dir -r requirements.txt

# 复制应用代码
COPY . .

# 创建目录
RUN mkdir -p data logs

EXPOSE 8000

CMD ["uvicorn", "main:app", "--host", "0.0.0.0", "--port", "8000"]
```

### 7.2 前端 Dockerfile

```dockerfile
# frontend/web/Dockerfile
FROM node:18-alpine as build

WORKDIR /app

COPY package*.json ./
RUN npm install

COPY . .
RUN npm run build

FROM nginx:alpine
COPY --from=build /app/dist /usr/share/nginx/html
COPY nginx.conf /etc/nginx/conf.d/default.conf

EXPOSE 80

CMD ["nginx", "-g", "daemon off;"]
```

### 7.3 Nginx 配置

```nginx
# frontend/web/nginx.conf
server {
    listen 80;
    server_name localhost;
    root /usr/share/nginx/html;
    index index.html;

    location / {
        try_files $uri $uri/ /index.html;
    }

    location /api {
        proxy_pass http://backend:8000;
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
    }
}
```

## 9. 常见问题

### 9.1 Python 相关

**Q: pip install 失败**
```bash
# 升级 pip
python -m pip install --upgrade pip

# 使用国内镜像
pip install -r requirements.txt -i https://pypi.tuna.tsinghua.edu.cn/simple
```

**Q: 导入模块报错**
```bash
# 确保在虚拟环境中
source venv/bin/activate
pip install -r requirements.txt
```

### 9.2 Node.js 相关

**Q: npm install 慢**
```bash
# 使用淘宝镜像
npm config set registry https://registry.npmmirror.com

# 或使用 cnpm
npm install -g cnpm --registry=https://registry.npmmirror.com
```

**Q: 构建失败**
```bash
# 清理缓存
rm -rf node_modules package-lock.json
npm install
```

### 9.3 数据库相关

**Q: SQLite 数据库被锁定**
```python
# 在 settings.py 中配置
engine = create_engine(
    DATABASE_URL,
    connect_args={"check_same_thread": False}
)
```
