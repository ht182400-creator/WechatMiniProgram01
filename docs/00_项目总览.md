# 股票量化分析与预测系统 V3.0 - 项目总览

## 📋 项目简介

股票量化分析与预测系统是一套完整的量化投资研究平台，支持A股市场数据采集、技术指标计算、策略回测和趋势预测。

## 🎯 核心功能

| 模块 | 功能 | 技术栈 |
|------|------|--------|
| 数据采集 | A股实时/历史数据 | AKShare/Baostock/Tushare |
| 本地缓存 | 数据缓存加速 | Parquet + SQLite |
| 技术指标 | MA/RSI/MACD/KDJ等 | pandas-ta |
| 策略回测 | 多策略验证 | 自研回测引擎 |
| 趋势预测 | ML涨跌预测 | scikit-learn |
| Web前端 | 图表可视化 | Vue 3 + ECharts |
| 微信端 | 移动端访问 | 原生小程序 |

## 🚀 技术选型

### 后端
- **Web框架**: FastAPI 0.109+
- **数据库**: SQLite (开发) / PostgreSQL (生产)
- **缓存**: Parquet 文件缓存
- **数据处理**: pandas, numpy
- **机器学习**: scikit-learn, xgboost

### 前端
- **框架**: Vue 3 + Composition API
- **UI库**: Element Plus 2.5+
- **图表**: ECharts 5.5+
- **构建**: Vite 5
- **状态**: Pinia
