## 2. 多数据源适配器实现

### 2.1 适配器模式设计

```python
# adapters/base_adapter.py
class BaseDataAdapter(ABC):
    """数据源适配器抽象基类"""
    
    @abstractmethod
    def connect(self) -> bool: ...
    
    @abstractmethod
    def get_daily_data(self, code, start_date, end_date) -> pd.DataFrame: ...
```

### 2.2 数据源管理器

```python
# adapters/data_source_manager.py
class DataSourceManager:
    """多数据源统一管理器"""
    
    def get_daily_data(self, code, start_date, end_date) -> pd.DataFrame:
        """按优先级尝试各数据源"""
        for name in ['akshare', 'baostock', 'tushare']:
            try:
                df = self.adapters[name].get_daily_data(...)
                if not df.empty:
                    return df
            except Exception:
                continue  # 尝试下一个数据源
        return pd.DataFrame()
```

### 2.3 优先级策略

1. **AKShare** - 默认首选，免费稳定
2. **Baostock** - 备用数据源
3. **Tushare** - 高频数据需求时启用

## 3. 缓存机制实现

### 3.1 Parquet 缓存

```python
# cache/parquet_cache.py
class ParquetCache:
    """Parquet 格式缓存"""
    
    def get(self, key: str) -> Optional[pd.DataFrame]:
        cache_path = self._get_cache_path(key)
        if not self._is_expired(cache_path):
            return pd.read_parquet(cache_path)
        return None
    
    def set(self, key: str, data: pd.DataFrame) -> bool:
        cache_path = self._get_cache_path(key)
        data.to_parquet(cache_path, index=True)
        return True
```

### 3.2 缓存策略

| 策略 | 说明 |
|------|------|
| 过期时间 | 默认 30 分钟 |
| 最大条目 | 500 条 |
| 自动清理 | 启动时清理过期文件 |

class BaseStrategy(ABC):
    """策略抽象基类"""
    
    @abstractmethod
    def generate_signals(self, data: pd.DataFrame) -> List[TradingSignal]:

### 5.2 支持的策略
